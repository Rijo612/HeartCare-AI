# ❤️ HeartCare AI

A Heart Disease Prediction Web Application built using Machine Learning and Streamlit.

## Features

- Heart Disease Prediction
- Random Forest Classifier
- BMI Calculator
- Health Score
- Interactive Charts
- Prediction History
- Download Report
- Feature Importance
- Medical Dashboard

## Technologies

- Python
- Streamlit
- Scikit-Learn
- Plotly
- Pandas
- NumPy
