# ❤️ HeartCare AI

HeartCare AI is a machine learning web application that predicts the risk of heart disease using patient health data. The application is built with Python, Streamlit, and Scikit-learn, providing an easy-to-use interface for risk prediction and visualization.

## 🚀 Features

- ❤️ Heart Disease Prediction
- 🤖 Random Forest Machine Learning Model
- 📊 Interactive Plotly Charts
- 📈 Prediction Confidence Score
- ❤️ Health Score
- 🧮 BMI Calculator
- 📋 Patient Summary
- 📜 Prediction History
- 📄 Download Prediction Report
- 💡 Health Recommendations
- 🎨 Modern Streamlit Dashboard

## 🛠️ Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Plotly
- Joblib
